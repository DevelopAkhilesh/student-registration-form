This is a student registration system.
A simple and responsive Student Registration System built using HTML, CSS, and JavaScript.
This project allows users to add, edit, delete, and manage student records with data stored in the browser using LocalStorage.
///////////////////////////////

🚀 Features

Add new student details

Edit existing student information

Delete student records with confirmation popup

Form validation for name, email, and contact number

Prevent duplicate student ID, email, and contact number

Responsive design for mobile, tablet, and desktop

Data persistence using LocalStorage

Error popup for invalid inputs
///////////////////////////////////

🛠 Technologies Used

HTML5 – Structure of the application

CSS3 – Styling and responsive design

JavaScript (Vanilla JS) – Application logic and DOM manipulation

LocalStorage API – Store student data in the browser

⚙️ How It Works

The user fills in the student details in the form.

When the form is submitted:

The input data is validated.

Duplicate entries are checked.

The student record is stored in LocalStorage.

The table automatically updates to show the stored records.

Users can edit or delete records anytime.
//////////////////////////////////

📱 Responsive Design

The application is fully responsive and works on:

Mobile devices

Tablets

Laptops

Desktop screens

The table automatically adjusts for smaller screens for better usability.
